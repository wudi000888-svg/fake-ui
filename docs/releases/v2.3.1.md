# fake-ui v2.3.1

v2.3.1 是 v2.3.0 的 Docker DNS 补丁版本。它修复安装脚本默认把国内 DNS 写入 Docker daemon 后，容器内节点解析可能受到 DNS 污染影响、导致节点无法正常使用的问题。

## Updates

- Docker daemon 默认 DNS 调整为 `1.1.1.1`、`8.8.8.8`、`9.9.9.9` 和 `208.67.222.222`。
- 安装脚本重新运行时会写入新的 `/etc/docker/daemon.json` 并重启 Docker，使 `xray`、`hysteria2`、`panel` 等容器使用新的解析器。
- 安装脚本、Docker Compose 默认版本、Windows 部署脚本和前端兜底版本号更新为 `v2.3.1`。

## Tests

- 本地：`python3 -m pytest tests/test_install_script.py tests/test_package_code.py -q`。
- 本地：`bash -n scripts/install-fresh-vps.sh`。
- 本地：`git diff --check`。

## Upgrade Notes

从 v2.3.0 可直接滚动升级到 v2.3.1。此版本不包含数据库 schema 迁移。

如果旧服务器已经由安装脚本生成 `/etc/docker/daemon.json`，升级后建议重新执行安装/部署流程，或手动确认该文件中的 `dns` 已更新为 v2.3.1 的海外公共解析器列表，然后重启 Docker。
