# fake-ui v3.0.2

v3.0.2 是一次面向公开交付的产品口径整理版本。底层仍然使用 Xray bridge / portal / Reality 完成跨境回连，但用户看到的功能统一命名为“本地服务发布”：把客户机器上的本地端口发布成海外 VPS 上的 HTTPS 域名、私有 TCP 或 SSH 入口。

| 类型 | 内容 |

## 使用方式

1. 公开 Web 服务：客户把子域名 A/AAAA 解析到 VPS，管理员在“本地服务发布”页新增服务，填写公网域名、本地地址和本地端口。
2. 私有 TCP/SSH：不需要域名，面板会明确标注 TCP 无需域名。
3. 每台后端机器下载一个后端客户端；共享 Agent 下多个服务会合并到同一个客户端配置。
4. 客户安装后打开 `http://127.0.0.1:19090/`，确认运行状态、服务探测和本地代理兼容状态。

## 升级说明

无需数据库 schema 迁移。已有 `tunnel_catalog`、Xray portal、Nginx HTTPS、配对 Agent 包和总 JSON 导出逻辑保持兼容。升级后主要变化是用户可见文案、发布说明和前端缓存版本。
